# World Reports

Два инструмента для анализа данных и инвестиционных решений.

## 1. World Bank Data Explorer (`worldbank_app.py`)

Анализ макроэкономических показателей Всемирного банка: ВВП, население, инфляция и другие.  
Двуязычный интерфейс (русский / English). Экспорт в PDF и CSV.

## 2. Портфельный контролёр с ИИ (`portfolio_controller/portfolio_controller.py`)

Прогноз цен на 7 дней с помощью модели **Prophet** (Meta).  
Сравнивает текущую цену с прогнозной вилкой и выдаёт цветной сигнал:

- 🟢 цена ниже прогноза → **покупка**
- 🟡 цена внутри диапазона → **держать**
- 🔴 цена выше прогноза → **продажа**

**Возможности:**
- выбор любых активов (акции, нефть, золото, индексы)
- настройка глубины истории (2–5 лет)
- интерактивные графики + сводная таблица

**Как запустить портфельный контролёр:**
```bash
cd portfolio_controller
pip install -r requirements.txt   (если ещё не установлены зависимости)
streamlit run portfolio_controller.py
